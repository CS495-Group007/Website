Simply Scansion

Group 007 - CS495 - Spring 2021

The goals of our project are to develop a fully functional web application that gameifies the practice of poetry scansion.
We want to deliver a fluid, seamless experience for students to study and for professors to monitor student progress. The mobile experience is our top priority, though we will need for the site to work well on laptops as well. The top priority is to make practicing scansion fun and simple.
Tools / APIs / Frameworks
  • React.JS Frontend
  • Node.JS + Express.JS Backend

User Experiences
  • Users can login to a personal account
  • Users can select Shakespearean Sonnets
  • Users can use color coding options to assign to sonnet blocks
  • Users can track their progress
  • Users can receive instructor feedback
Administrator Experiences
  • Admins can import new sonnets
  • Admins can assign sonnets to students
  • Admins can monitor individual and class-wide progress
